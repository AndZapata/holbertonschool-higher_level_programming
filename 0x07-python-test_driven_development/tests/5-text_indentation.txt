# text_indentation.txt
===============================
 How to Use text_indentation.py
===============================

This library is very simple, since it only has one function called
``text_indentation()``.

Numbers
=======

``text_indentation()``

::

    >>> text_indentation = __import__('5-text_indentation').text_indentation
    >>> text_indentation("Holberton.")
    5
    >>> text_indentation(-3, 10)
    7
    >>> text_indentation(40, -5)
    35
    >>> text_indentation(-40, -5)
    -45

It also works with floating-point values.

::

    >>> text_indentation(2.0, 3)
    5

Non-Numbers
===========

Raise error message in cases that one of the variables are not integers or
floats

::

    >>> text_indentation('a', 3)
    Traceback (most recent call last):
    TypeError: a must be an integer

    >>> text_indentation("Holberton", 2)
    Traceback (most recent call last):
    TypeError: a must be an integer

    >>> text_indentation(9, "School")
    Traceback (most recent call last):
    TypeError: b must be an integer

    >>> text_indentation(100, 'G')
    Traceback (most recent call last):
    TypeError: b must be an integer
